# PhishGuard

PhishGuard is a full-stack web application that helps users detect phishing attacks in messages, URLs, and emails using AI-inspired analysis.

---

## 🚀 Features

- 🔐 User Authentication (Login & Signup)
- 🧪 Message Scanner (detects scam keywords)
- 🔗 URL Scanner (checks suspicious links)
- 📧 Email Analyzer (detects phishing emails)
- 🤖 AI-based Risk Detection (risk score + explanation)
- 📊 Dashboard with scan history
- 📘 Awareness tips for cybersecurity

---

## 🛠️ Tech Stack

- Frontend: React.js
- Backend: Firebase (Authentication + Firestore)
- Styling: CSS / Modern UI Design

---

## ⚙️ How it Works

- Users log in securely
- Input message, URL, or email
- System analyzes content using keyword-based logic
- Displays:
  - Status (Safe / Scam)
  - Risk Score (%)
  - Explanation
- Saves results in database for history tracking
